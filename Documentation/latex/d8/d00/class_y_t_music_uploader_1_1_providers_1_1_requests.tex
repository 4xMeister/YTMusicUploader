\doxysubsection{Detailed Description}
Http\+Web\+Request P\+O\+ST request to send to You\+Tube to check if the user\textquotesingle{}s is authenticated (signed in) by determining if a generic request is successful given the current authentication cookie value we have stored. 

You\+Tube music Http\+Web\+Request abstract class

Http\+Web\+Request P\+O\+ST request to send to You\+Tube to upload a music file.

Http\+Web\+Request P\+O\+ST request to send to You\+Tube to determine if the song about to be uploaded already exists It may request a little attention depending on the results, seen as though we\textquotesingle{}re trying to match a song based on artist, album and track name, which may be slightly different in the uploading music file\textquotesingle{}s meta tags. Currently it uses a Levenshtein distance fuzzy logic implementation to check similarity between strings and is considered a match if it\textquotesingle{}s above 0.\+75.

In this case, we\textquotesingle{}re actually perform a request for personally uploaded music files
